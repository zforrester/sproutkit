# My Site

## Setup
### External Dependencies
- Install NPM
  https://nodejs.org/en/
- Install Gulp globally
  `npm install --global gulp-cli`
  https://github.com/gulpjs/gulp/blob/master/docs/getting-started.md

### Initiate
- Run
  `npm install`

- Run
  `gulp`

  The project should now be running and ready to be worked on
