## Set up

Typographic properties are defined in `tokens/typography.json`. These propertiers include `sizes`, `families`, `weights`, `line-heights`, and `letter-spacing`. You can modify these property definitions or replace them with your own.

## Usage

The properties can accessed by using a function name that matches the CSS property name.

```
.headline {
	font-size: font-size()`
	font-family: font-family()`
	font-weight: font-weight()`
	line-height: line-height()`
	letter-spacing: letter-spacing()`
}
```

Alternatively, there is a `@mixin font()` that provides a short cut for these propeties.

```
// SCSS
.headline {
  @include font(32, 'bold', 'primary');
}

// CSS
.headline {
	font-size: 32px;
	font-weight: 600;
	font-family: 'Avenir Next';
}
```
